## 💁 Service introduce
지역 생활 커뮤니티 서비스 시장을 이끌고 있는 당근마켓 앱을 모방하여 개발함으로써 앱 개발 역량을 키우는 것을 목표로 해당 프로젝트를 기획하게 되었습니다.

- BSImagePicker
